# Super-Store-Analysis-BI-Project

## Project Overview

This project is a comprehensive Super Store Sales Analysis Dashboard built using Power BI. The dashboard provides valuable insights into sales performance, profit trends, customer behavior, and regional analysis to support business decision-making.

## Features

* Sales and Profit Analysis
* Region-wise and State-wise Performance
* Category and Sub-category Insights
* Customer Segment Analysis
* Interactive Filters and Visualizations
* KPI Tracking Dashboard

## Tools & Technologies

* Power BI
* DAX
* Excel / CSV
* Data Cleaning & Transformation
* Data Visualization

## Key Insights

* Identified top-performing regions and products
* Analyzed profit trends across categories
* Discovered customer purchasing patterns
* Created interactive visual reports for better decision-making

## Project Outcome

The project demonstrates skills in:

* Business Intelligence
* Data Visualization
* Dashboard Development
* Analytical Thinking
* Storytelling with Data
